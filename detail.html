Tentang Aplikasi

Aplikasi ini dikembangkan sebagai Progressive Web App (PWA) yang memungkinkan pengguna untuk membuat, mengedit, dan menyimpan profil secara lokal di browser. Tujuan utamanya adalah memberikan pengalaman yang cepat, ringan, responsif, serta dapat berjalan secara offline.

Teknologi yang Digunakan
  • HTML5 – Struktur markup halaman.
  • CSS3 – Gaya dan tata letak visual.
  • Bootstrap 5 – Komponen responsif & sistem grid.
  • JavaScript ES6+ – Logika interaktif & asinkron.
  • IndexedDB – Penyimpanan data profil secara lokal.
  • localStorage – Alternatif penyimpanan jika IndexedDB tidak tersedia.
Fitur Responsif
  • Tampilan adaptif di semua ukuran layar (mobile, tablet, desktop).
  • Penggunaan grid system seperti col-md-4 & col-md-8.
  • Meta tag viewport untuk skala layar yang sesuai.
Gambar Profil Dinamis
  • Pengguna dapat mengunggah gambar profil sendiri.
  • Disimpan secara lokal menggunakan IndexedDB atau localStorage.
  • Efisien berkat atribut loading="lazy".
PWA (Progressive Web App)
  • Service Worker mendukung mode offline & caching.
  • manifest.json untuk instalasi ke home screen.
  • Sumber daya dari CDN disimpan untuk akses cepat berikutnya.
Fungsi loadProfile()
  • Mengambil data secara asinkron dari IndexedDB.
  • Fallback otomatis ke localStorage jika gagal.
  • Menampilkan nama, username, deskripsi, gambar, dan statistik GitHub pengguna.
  • Gunakan fetch ke API publik GitHub.
