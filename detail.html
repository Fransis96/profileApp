Tentang Aplikasi

Aplikasi ini dikembangkan sebagai Progressive Web App (PWA) yang memungkinkan pengguna untuk membuat, mengedit, dan menyimpan profil secara lokal di browser. Tujuannya adalah memberikan pengalaman yang cepat, ringan, responsif, serta dapat berjalan secara offline.

Teknologi yang Digunakan
HTML5 – Struktur markup halaman
CSS3 – Gaya dan tata letak visual
Bootstrap 5 – Komponen responsif & grid
JavaScript ES6+ – Logika interaktif
IndexedDB – Penyimpanan lokal
localStorage – Alternatif penyimpanan
Fitur Responsif
  • Tampilan adaptif semua layar
  • Sistem grid col-md-4 dan lainnya
  • Meta tag viewport
Gambar Profil Dinamis
  • Dapat unggah gambar sendiri
  • Disimpan di IndexedDB
  • Menggunakan loading="lazy"
Progressive Web App (PWA)
  • Service Worker untuk offline
  • Manifest untuk instalasi
  • Cache CDN otomatis
Fungsi loadProfile()
  • Ambil data dari IndexedDB
  • Fallback ke localStorage
  • Tampilkan nama, gambar, stats
  • Gunakan fetch ke GitHub
